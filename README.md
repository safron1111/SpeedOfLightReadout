# SpeedOfLightReadout (or cSpeed for short)
Simple KSP mod that shows a window when you press Alt+C or the toolbar button that gives info on your vessel or target vessel in relation to light speed. \
<img width="338" height="426" alt="image" src="https://github.com/user-attachments/assets/cd7370ba-f79e-48d1-89e8-01e533b8094f" />

## Features
Can modify the value of the speed of light using a multiplier (for those who want to 'kerbalize' light speed). \
<img width="312" height="76" alt="image" src="https://github.com/user-attachments/assets/0a23d585-fbb0-4763-9390-a26780fb7361" /> 

Can view your vessel's current speed in a fractional or percentile way. \
<img width="256" height="100" alt="image" src="https://github.com/user-attachments/assets/2f0a1765-b801-4935-ae9f-a54a12cbca0c" /> 

Can see your distance to your orbiting body in center-to-center or altitude. \
Can change what light-based unit to represent distance in. \
Can make light-based units based on the current homeworld rather than real-life (Kerbin day-length vs Earth day-length, etc). \
<img width="312" height="165" alt="image" src="https://github.com/user-attachments/assets/3c2c59d1-80cb-4f30-ad2c-6ddad2ff2961" /> 

Can view a target-based readout window. \
<img width="326" height="295" alt="image" src="https://github.com/user-attachments/assets/b98541db-c7cb-49bb-8dc2-98117180ada7" /> 

## Instructions
1. Enter flight mode on any vessel
2. Press Alt + C or the toolbar button labeled cSpeed
